## 🚀 About Me

I'm **Cristian Perez**, a **Full-Stack Developer, AI Engineer, and Data Analyst** based in the **Dominican Republic** with **6+ years of experience** building modern digital products — from polished user interfaces to intelligent, AI-powered systems.

I specialize in creating scalable, user-focused applications across the full stack using **React, Next.js, Angular, TypeScript, Node.js, NestJS, and GraphQL**. In recent years, I've expanded deep into **AI engineering** — building intelligent applications, automation pipelines, custom chat agents, and LLM-powered tools using frameworks like **LangChain, OpenAI API, Anthropic Claude**, and the **Model Context Protocol (MCP)**.

I also bring a strong foundation in **data analysis**, using **Python, Pandas, Matplotlib, and Tableau** to uncover insights and support smarter business decisions.

- 🎯 Clean code, maintainable architecture, and genuine business value
